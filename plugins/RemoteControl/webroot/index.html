API Explorer

																

														

														
														

																

																

																		The API Explorer is a built-in testing tool that allows you to interact directly with 
																		Stellarium's HTTP API without writing any code. Every service and operation available 
																		through the Remote Control plugin is listed here and can be tested with a single click.
																

																

																		This is the same API that powers the entire Remote Control interface. By exploring it here,
																		you can understand how each panel communicates with Stellarium and learn to build your own 
																		integrations, scripts, or external applications.
																

														

														
														

																
How to Use

																
    
																		
  1. 
																				Select a service from the sidebar on the left. 
																				Each service represents a category of operations (e.g., Main Service for core functions, 
																				Object Service for celestial object searches, Script Service for script management).
																		
    2. 
																		
  2. 
																				Click on an operation within the expanded service. 
																				The operation panel will open on the right showing the endpoint path, HTTP method, 
																				description, and all available parameters.
																		
    3. 
																		
  3. 
																				Fill in the required parameters (marked with a red asterisk). 
																				Optional parameters can be left empty - the API will use default values when available.
																				Hover over parameter descriptions for hints about expected values and formats.
																		
    4. 
																		
  4. 
																				Click "Execute Request" to send the request to Stellarium 
																				and view the raw response. The HTTP status code and response body are displayed 
																				immediately in the response area below.
																		
    5. 
																		
  5. 
																				Use "Generate cURL" to get a copy-paste ready command line 
																				equivalent. This is invaluable for scripting, automation, or debugging from a terminal.
																		
    6. 
																

														

														
														

																
Key Features

																
    
																		
  • 
																				Live Request Execution: Every request is sent directly to your 
																				running Stellarium instance. Changes made here are real and immediate.
																		
    • 
																		
  • 
																				Response Inspection: View raw HTTP responses including status codes, 
																				headers, and body content. JSON responses are automatically formatted for readability.
																		
    • 
																		
  • 
																				cURL Export: Any request can be exported as a cURL command, making 
																				it easy to integrate into shell scripts, cron jobs, or external automation tools.
																		
    • 
																		
  • 
																				Service Filtering: Use the filter box at the top of the sidebar to 
																				quickly find services or operations by name. This is especially useful when working 
																				with the full set of available endpoints.
																		
    • 
																		
  • 
																				Copy to Clipboard: Both response data and cURL commands can be 
																				copied with a single click for use in other applications.
																		
    • 
																

														

														
														

																
Available Services

																

																		

																				M
																				Main Service
																				Core functions: time, view, focus, movement, status, window
																		

																		

																				O
																				Object Service
																				Search and retrieve celestial object data
																		

																		

																				L
																				Location Service
																				Set observer location, search locations, manage planets
																		

																		

																				R
																				Script Service
																				Run, stop, and manage Stellarium scripts
																		

																		

																				A
																				StelAction Service
																				Execute and toggle Stellarium actions
																		

																		

																				P
																				StelProperty Service
																				Read and modify Stellarium properties
																		

																		

																				S
																				Location Search
																				Search location database by name or coordinates
																		

																		

																				W
																				SIMBAD Service
																				Query the online SIMBAD astronomical database
																		

																		

																				V
																				View Service
																				List landscapes, sky cultures, and projections
																		

																

														

														
														

																
Common Use Cases

																
    
																		
  • Debugging: Verify that Stellarium is responding correctly before writing scripts or integrating with other tools.
    • 
																		
  • Learning the API: Understand the structure of API requests and responses by experimenting with different parameters and observing the results.
    • 
																		
  • Integration Development: Test API calls before implementing them in your own applications, web services, or automation scripts.
    • 
																		
  • Quick Actions: Execute one-off commands without navigating through the full interface panels.
    • 
																		
  • Script Development: Test individual API operations to understand their behavior before combining them into complex scripts.
    • 
																

														

														
														

																
Pro Tip

																

																		Combine the API Explorer with the Code Generator tab. Once you find the 
																		right API operation and parameters here, switch to the Code Generator to produce ready-to-use 
																		code in cURL, Python, JavaScript, or Stellarium Script format. You can then insert that code 
																		directly into the Script Editor for further customization.
