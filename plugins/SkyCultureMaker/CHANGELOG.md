# Sky Culture Maker Plugin - Changelog

## Version 1.0.1

### New Features

* Added option to automatically merge lines into polylines on export (#4563)
* Moved constellation description input fields to constellation editor (#4594)

### Bug Fixes

* Erased lines are still exported (#4595)
* Drawn constellations are not reset after "Export and Exit" (#4557)
* "Export and Exit" exits even when export failed, deleting all unsaved progress (#4580)
* Stop drawing line key (Double Right Click) only works when Star/DSO is selected (#4557)
* Not all SC description input fields that are marked required are actually checked (#4557)
* No formal checks for 'Constellations' input field in SC description (#4557)
